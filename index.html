Tomoko

Partial updates in REST API made easy

Tomoko is a Java 8 library that allows developers to implement REST PATCH requests in accordance with JSON Patch RFC 6902 standard.

As a developer you just have to attach handlers (lambdas) for each valid path. These handlers will receive path parameters and the parsed operation value.

Tomoko takes care of parsing the whole request, checking if the request is valid according to the defined model, finding handlers and invoking them.

It supports Jackson and Gson libraries so you don't have to parse JSON Patch requests yourself.
